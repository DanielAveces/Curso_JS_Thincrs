Matriz de objetos.

El desafío consiste en escribir una función que tome una matriz de objetos y devuelva un nuevo arreglo que contenga solo los objetos que cumplan con ciertas condiciones. El objeto que se ingrese en dicha matriz de objetos deberá ser un objeto persona, el cual tendrá las siguientes propiedades: • Age – Number • Name – String • Gender – String • Weight – Number • Height – Number • HaveWork – Boolean • FavoriteColors – Array Los filtros que se generen deberán de ser a elección del alumno, pero tienen que ser 3 opciones, es decir se pueden escoger entre la lista, la única condición es que al menos sean dos tipos de datos diferentes. Para todo esto se deberá de generar una aplicación de consola que permita tanto ingresar personas nuevas, como filtrar a las mismas.
