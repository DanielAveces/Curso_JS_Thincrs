Solicitudes HTTP

El desafío consiste en escribir una función que realice una solicitud HTTP a una API y muestre los resultados en una página web. La API para consultar la información deberá de ser https://pokeapi.co/ y se deberá de realizar con la librería de axios o el método fetch, a elección del alumno. Se deberá de escoger 3 API a llamar a elección del alumno con botones diferentes. Se obtienen puntos extras si se realiza un paginado o un scroll infinito para cualquiera de las 3 API. **Se puede pedir ayuda al instructor para este apartado, ya que es un tema un poco más avanzado.**
